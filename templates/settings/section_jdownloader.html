JDownloader

Envoie les liens directs vers ton JDownloader via My.JDownloader, dans un dossier choisi automatiquement selon le type de média.

Connexion

Renseigne l'email et le mot de passe du compte lié à ton JDownloader, teste la connexion pour détecter tes appareils, choisis celui à piloter, puis enregistre. Le mot de passe est conservé localement dans config.json (nécessaire pour envoyer les liens).

Dossiers de destination

Chemins tels que vus par la machine où tourne JDownloader. Coche « sous-dossier » pour créer un dossier au nom du média sous le dossier de base ; décoche pour envoyer directement dans un dossier existant.
